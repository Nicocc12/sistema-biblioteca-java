# Sistema de Gestion de Catalogo y Prestamos SaberyFuturo
Desarrollé una solución de software de escritorio en Java para automatizar la gestión de la Biblioteca 'Saber y Futuro', reemplazando un sistema obsoleto basado en cuadernos y planillas de Excel. El objetivo principal fue centralizar la administración de socios y materiales, garantizando la integridad de los datos.
